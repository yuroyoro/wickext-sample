WickeXt Helpers - JQueryのAPIをWikcetへ

TraversingHelp.eq(int position) - $()でマッチした要素から指定Indexの要素を返します。 
JsQuery jsq = new JsQuery();
jsq.$(".foo")
   .chain(TraversingHelper.eq(1))
   .chain(CssHelper.css(new Options().putLiteral("border", "1px solid red")));
jsq.contribute(this);

2番目の要素だけ赤くなるはず。

class=foo class=foo class=foo

TraversingHelp.siblings(String expression) - 指定したセレクタの兄弟要素を返します。 
JsQuery jsq = new JsQuery();
jsq.$("#foo")
   .chain(TraversingHelper.siblings(".style-me"))
   .chain(CssHelper.css(
  	new Options().putLiteral("border", "1px solid red")
   ));
jsq.contribute(this);

3番目の要素が赤くなるはず

id=foo Sibling of #foo Sibling of #foo with class=style-me

AttributesHelper.addClass(String className) - cssクラスを追加します。 
JsQuery jsq = new JsQuery();
jsq.$(".foo2")
   .chain(TraversingHelper.siblings(".style-me"))
   .chain(AttributesHelper.addClass("you-are-now-styled"));
jsq.contribute(this);

class=foo2 Just a span ! This element now holds class=style-me & class=you-are-now-styled

CssHelper.css(String name, String value) - CSSの要素を追加します。 
JsQuery jsq = new JsQuery();
jsq.$(".foo-important")
   .chain(CssHelper.css("border", "1px solid red"));
jsq.contribute(this);

1番目と3番目が赤くなるはず。

class=foo2 Just a span ! This element now holds class=style-me & class=you-are-now-styled
